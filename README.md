# hyrule-jobs
Project to learn basics of Vue.js. Followed this tutorial: https://youtu.be/JfI5PISLr9w

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
